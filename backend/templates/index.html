Available API Endpoints

Authentication

Manage JWT tokens and user authentication

  • Obtain JWT Token POST
  • Refresh Token POST
  • Verify Token POST
Users

Create and manage users with different roles

  • User List (Admin only) GET
  • Create User POST
  • User Profile GET
  • Update Profile PATCH
  • Change Password POST
Events

Create and manage sporting events

  • Event List GET
  • Create Event POST
  • Event Details GET /:id
  • Update Event PUT /:id
  • Delete Event DELETE /:id
Teams

Manage teams participating in events

  • Team List GET
  • Create Team POST
  • Team Details GET /:id
  • Update Team PATCH /:id
  • Delete Team DELETE /:id
Games

Manage games and matches

  • Game List GET
  • Create Game POST
  • Game Details GET /:id
  • Update Game Status PATCH /:id
Scores & Leaderboards

Track scores and view leaderboards

  • Scores List GET
  • Record Score POST
  • Event Leaderboard GET
  • Team Leaderboard GET
