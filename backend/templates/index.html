Available API Endpoints

Authentication

Manage JWT tokens and user authentication

  • Obtain JWT Token POST
  • Refresh Token POST
  • Verify Token POST
Users

Create and manage users with different roles

  • User List (Admin only) GET
  • Create User POST
  • User Profile GET
  • Update Profile PATCH
  • Change Password POST
Events

Create and manage sporting events

  • Event List GET
  • Create Event POST
  • Event Details GET /:id
  • Update Event PUT /:id
  • Public Events GET
Sport Events

Manage sport-specific events within main events

  • Sport Events List GET
  • Create Sport Event POST
  • Sport Event Details GET /:id
  • Update Sport Event PUT /:id
  • Public Sport Events GET
  • Update Status PATCH /:id/update-status
Teams

Manage teams participating in events

  • Team List GET
  • Create Team POST
  • Team Details GET /:id
  • Update Team PATCH /:id
  • Team Members GET /:id/members
  • Add Team Member POST /:id/members
Games

Manage games and matches

  • Game List GET
  • Create Game POST
  • Game Details GET /:id
  • Update Game Status PATCH /:id/status
  • Record Score POST /:id/score
  • Game Schedule GET /schedule
Scores & Leaderboards

Track scores and view leaderboards

  • Scores List GET
  • Record Score POST
  • Event Leaderboard GET
  • Team Leaderboard GET
  • Sport Type Leaderboard GET
Participants

Manage event participants and registrations

  • Participants List GET
  • Register Participant POST
  • Participant Details GET /:id
  • Update Registration PATCH /:id
  • Cancel Registration DELETE /:id
Reports

Access sports event analytics and reports

  • Event Summary GET /events/:id/summary
  • Participation Report GET /participation
  • Performance Metrics GET /performance
  • Sport Analysis GET /sports/analysis
  • Export Results GET /export/:format
